# Portfolio de Mamadou Kouma

Portfolio moderne et attractif créé avec React, TypeScript et TailwindCSS v4.

## 🚀 Démarrage rapide

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Build pour la production
npm run build
```

## 📁 Structure du projet

```
src/
├── components/
│   ├── Portfolio.tsx      # Composant principal du portfolio
│   └── Portfolio.css      # Animations personnalisées
├── App.tsx                # Point d'entrée de l'application
├── App.css                # Styles globaux
├── index.css              # Configuration Tailwind
└── main.tsx               # Montage React
```

## ✨ Fonctionnalités

- ✅ Design moderne avec dégradés purple/pink
- ✅ Navigation fixe responsive avec menu mobile
- ✅ Animations au scroll pour tous les blocs
- ✅ Cartes interactives avec effets hover
- ✅ Section Hero avec avatar animé
- ✅ Grille de compétences par catégorie
- ✅ Timeline d'expérience professionnelle
- ✅ Showcase de projets avec icônes
- ✅ Section contact avec informations
- ✅ Footer élégant

## 🎨 Technologies utilisées

- React 19
- TypeScript
- TailwindCSS v4
- Vite 8
- PostCSS

## 📱 Sections du portfolio

1. **Accueil** - Présentation avec CTA
2. **Compétences** - Langages, frameworks, outils
3. **Expérience** - Parcours professionnel
4. **Projets** - Projets principaux réalisés
5. **Contact** - Email et téléphone

## 🎯 Animations incluses

- Animation de flottement pour l'avatar
- Apparition progressive des sections au scroll
- Effet de scale sur les cartes au survol
- Transitions de couleur fluides
- Effets de brillance sur les boutons

## 🌐 Accès

Le serveur de développement est accessible sur : http://localhost:5175/

## 📝 Personnalisation

Pour personnaliser le portfolio, modifiez le fichier `src/components/Portfolio.tsx` :

- Modifier les données dans les objets `skills`, `experiences`, `projects`
- Ajuster les couleurs dans les classes Tailwind
- Personnaliser les animations dans `Portfolio.css`
